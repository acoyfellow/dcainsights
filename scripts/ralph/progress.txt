## Codebase Patterns
- Blog posts: src/routes/blog/{slug}/+page.svelte with script, svelte:head, main, article structure
- Use Lucide icons, Tailwind CSS, Svelte 5 $state/$derived for interactivity
- Sitemap: src/routes/sitemap.xml/+server.ts - add new routes to routes array
- Blog index: src/routes/blog/+page.svelte - add posts to posts array
- Build: ~/.bun/bin/bun tsc && ~/.bun/bin/bun run build
- Keep blog posts focused, use interactive elements (calculators, tables) for engagement
